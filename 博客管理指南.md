# 博客管理指南

## 📝 项目结构

```
claude text/
├── posts/                    # 博客文章目录（重要）
│   ├── hello-world.mdx      # 示例文章1
│   └── nextjs-best-practices.mdx  # 示例文章2
├── config.ts                 # 网站配置文件
├── app/                      # Next.js 应用页面
├── components/               # React 组件
├── public/                   # 静态资源
└── types/                    # TypeScript 类型定义
```

## 🚀 快速开始

### 1. 自定义网站信息

编辑 `config.ts` 文件来个性化您的博客：

```typescript
export const siteConfig = {
  // 基本信息
  title: '您的博客标题',           // 改成您的博客名称
  author: '您的名字',             // 改成您的名字
  description: '您的博客描述',     // 改成您的博客简介
  url: 'https://your-domain.com', // 您的域名（部署时）

  // 社交链接
  social: {
    github: 'https://github.com/您的GitHub用户名',
    twitter: 'https://twitter.com/您的Twitter用户名',
    email: '您的邮箱@example.com',
  },
}
```

### 2. 添加新文章

#### 步骤1：创建 MDX 文件

在 `posts/` 目录下创建新的 `.mdx` 文件，文件名使用英文，例如：
- `my-first-post.mdx`
- `react-tutorial.mdx`
- `web-development-tips.mdx`

#### 步骤2：编写文章元数据

每个 MDX 文件开头必须包含 YAML 前置元数据：

```yaml
---
title: "文章标题"
description: "文章简介，会显示在文章列表中"
date: "2024-01-20"        # 发布日期
author: "作者名"           # 可以覆盖 config.ts 中的作者
tags: ["标签1", "标签2", "标签3"]  # 用于分类和筛选
pinned: true              # 可选：是否置顶文章
cover: "https://example.com/cover-image.jpg"  # 可选：封面图片
---
```

#### 步骤3：编写文章内容

使用 Markdown 语法编写内容，支持：

- **标题**: `# 一级标题` `## 二级标题`
- **段落**: 直接写文本
- **代码块**:
  ```javascript
  const code = "hello world";
  ```
- **列表**:
  - 项目1
  - 项目2
- **链接**: [链接文本](https://example.com)
- **图片**: ![图片描述](https://example.com/image.jpg)

### 3. 文章模板

复制以下模板来快速创建新文章：

```mdx
---
title: "新文章标题"
description: "这篇文章是关于..."
date: "2024-01-20"
author: "您的名字"
tags: ["标签1", "标签2"]
pinned: false
cover: "https://images.unsplash.com/photo-xxx?w=1200&h=630&fit=crop&crop=entropy&auto=format"
---

# 文章标题

这里是文章的引言...

## 第一节

内容...

## 第二节

内容...

### 子标题

更详细的内容...

## 总结

总结要点...
```

## 🎨 自定义样式

### 修改主题颜色

编辑 `tailwind.config.js`：

```javascript
theme: {
  extend: {
    colors: {
      primary: '#您的主题色',      // 主色调
      secondary: '#您的辅助色',    // 辅助色
    },
  },
}
```

### 修改字体

在 `app/globals.css` 中更新字体：

```css
font-family: ['您喜欢的字体', 'Noto Sans SC', 'system-ui', 'sans-serif'];
```

## 📸 添加图片

### 1. 使用本地图片

将图片放在 `public/` 目录下：

```
public/
├── images/
│   ├── my-photo.jpg
│   └── post-covers/
│       └── article-cover.jpg
```

在文章中引用：
```mdx
![图片描述](/images/my-photo.jpg)
```

### 2. 使用在线图片

推荐使用 Unsplash、Pexels 等免费图片网站：

```mdx
![图片描述](https://images.unsplash.com/photo-xxx?w=800&h=400&fit=crop)
```

## 🔧 高级功能

### 1. 文章置顶

在元数据中添加 `pinned: true`，文章会显示在列表顶部。

### 2. 自定义封面

在元数据中设置 `cover` 字段，支持：
- 本地图片：`/images/cover.jpg`
- 在线图片：`https://example.com/image.jpg`

### 3. 标签管理

- 标签用于文章分类和筛选
- 首页会自动显示所有标签
- 点击标签可以筛选相关文章

## 📝 写作技巧

### 1. SEO 优化

- 标题要包含关键词
- 描述要吸引人点击
- 合理使用标签
- 添加内部链接

### 2. 内容格式

- 使用清晰的标题结构
- 段落不要太长
- 适当使用列表和代码块
- 添加相关图片

### 3. 发布频率

建议保持规律的发布频率：
- 每周1-2篇文章
- 质量比数量重要
- 建立写作习惯

## 🛠️ 常见问题

### Q: 如何修改文章顺序？
A: 文章按日期排序，最新的在前。设置 `pinned: true` 可以置顶。

### Q: 如何添加自定义页面？
A: 在 `app/` 目录下创建新的文件夹和 `page.tsx` 文件。

### Q: 如何备份文章？
A: 定期备份 `posts/` 目录和 `config.ts` 文件。

### Q: 如何添加评论功能？
A: 配置 `config.ts` 中的 `giscus` 设置，使用 GitHub Discussions。
